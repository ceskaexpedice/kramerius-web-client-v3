How can I log in to the MZK Digital Library?

You can only log in to the MZK Digital Library via eduID. Institutions that are not part of eduID can create an NDK ID, however this login only works in the National Digital Library operated by the National Library of the Czech Republic.

A list of institutions with information on which login method to use can be found here. The last column indicates whether the institution allows remote registration. For more information, see the complete FAQ.

Logging in via eduID:

  • check the box to agree to the terms and then click the blue button "Log in with library account/eduID"
  • select your institution from the list
  • after selecting your institution you will be redirected to its login page
  • log in with the credentials you use at your institution
  • if only an empty list appears after clicking the Log in with eduID button, simply refresh the page using the F5 key or the browser's refresh icon
  • always log out of the application when you are done, using the link in the top right corner
